Theming

From Theme

We use the react-css-theme-switcher1 hook to manage themes.

import { ThemeSwitcherProvider } from "react-css-theme-switcher";

It works by looking for css to load in ./public/{name}.css

Updates

Dendron listens to the ThemeChange event from the plugin.

This is currently broadcasted when we load the webview initially here.

  • NOTE: currently, in order to broadcast theme changes, a user needs to run ReloadWindow. We have an action item to change this dynamcally.

On the webview side, Dendron uses the ThemeSwitcherProvider to pass the current theme to downstream components
