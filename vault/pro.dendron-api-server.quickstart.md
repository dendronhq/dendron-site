## Summary

### Start Server

```sh
yarn start:integ
```

### Initialize the server
```sh
API_SERVER="localhost:3005"
WORKSPACE=""
curl --location --request GET "$WORKSPACE_URL/api/workspace/initialize" \
--header 'Content-Type: application/json' \
--data-raw '{
    "uri": "$WORKSPACE"
}'
```
